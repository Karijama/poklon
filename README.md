# Go Youth
=======
# Описание проекта

Веб-портал, который имеет в себе функционал записи на мероприятия, новостную ленту, обсуждения, блоги и систему геймификации.

# Стек технологий

Docker Desktop, GitBash, Figma вместе с React.JS, Tailwind CSS, Firebase
